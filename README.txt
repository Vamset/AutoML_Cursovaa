# Прогнозирование продаж с использованием модели AutoML (FEDOT)

## Описание

В этом репозитории представлен проект, реализующий задачу прогнозирования временных рядов для продаж товаров в различных магазинах с использованием модели AutoML FEDOT.

## Структура репозитория

- **data/**: Папка с исходными данными:
  - `holidays_events.csv`: Данные о праздниках.
  - `oil.csv`: Данные о ценах на нефть.
  - `stores.csv`: Данные о магазинах.
  - `sample_submission.csv`: Пример отправки.
  - `test.csv`: Данные для тестирования.
  - `submission.csv`: Результаты предсказания.
  
- **logs/**: Папка для логов выполнения (например, `debug_logs.log`).

- **src/**: Папка с исходным кодом:
  - `main.py`: Основной скрипт для загрузки данных, обучения модели и предсказания.

- **README.md**: Описание проекта и инструкции.

- **requirements.txt**: Список зависимостей для проекта.

## Установка зависимостей

Для установки зависимостей выполните следующую команду:

pip install -r requirements.txt


## Запуск проекта

1. Скачайте или клонируйте репозиторий.
2. Убедитесь, что все данные находятся в папке `data/`.
3. Запустите основной скрипт:

python src/main.py
